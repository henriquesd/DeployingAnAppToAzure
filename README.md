# BookStore
Web API Project using .NET 5

## Technologies
- .NET 5
- Entity Framework 5
- Fluent API
- AutoMapper
- Swagger
- SQL Server

### Unit tests
- xUnit
- Moq
- SQLite In-Memory database

## Architecture
- 3 Layers:
  - Application layer (API)
    - Controllers
    - Dtos
  - Domain layer
    - Models
    - Interfaces
    - Services
  - Infrastructure layer
    - Repository Pattern
